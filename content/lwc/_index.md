# Lightning Web Components (LWC)

Lightning Web Components is a modern JavaScript framework for building user interfaces on the Salesforce platform. It leverages web standards and provides a lightweight, performant way to create custom components.

## Why LWC?

- **Standards-Based**: Built on modern web standards (Web Components, ES6+)
- **Performance**: Lightweight framework with better performance than Aura
- **Developer-Friendly**: Familiar programming model for web developers
- **Reusable**: Create modular, reusable components
- **Secure**: Built-in security features and Lightning Locker

## Core Concepts

- **Components**: Self-contained units of functionality
- **Composition**: Build complex UIs by combining simple components
- **Data Binding**: Reactive properties and wire adapters
- **Events**: Component communication through events
- **Lifecycle Hooks**: Control component behavior at different stages
